# StockEarningAnalysis

## Overview

StockEarningAnalysis is a Java research project that studies analyst
earnings forecast accuracy for large-cap and small-cap U.S. firms from
2015 to 2025.

The project compares forecast accuracy across three periods:

-   **2015-2019:** Pre-pandemic period
-   **2020-2022:** COVID-19 pandemic period
-   **2023-2025:** AI era period



The analysis supports the research paper:

> **Has Earnings Forecast Accuracy Improved in the AI Era? Evidence from
> S&P 500 and Russell 2000 Firms**

## Forecast Error Measures

### Price-Scaled Absolute Forecast Error

`|Actual EPS - Forecast EPS| / Previous Closing Price × 100`

Lower values indicate higher forecast accuracy.

### Price-Scaled Signed Forecast Error

`(Actual EPS - Forecast EPS) / Previous Closing Price × 100`

-   Positive values indicate actual earnings exceeded analyst forecasts.
-   Negative values indicate actual earnings were below analyst
    forecasts.

## Statistical Methods

The project calculates:

-   Mean
-   Standard deviation
-   Median
-   Mean Absolute Error (MAE)
-   Root Mean Squared Error (RMSE)
-   25th percentile
-   75th percentile
-   1% winsorized mean

Statistical tests include:

-   One-sample t-tests for forecast bias
-   Welch two-sample t-tests for comparisons across groups

The project uses **1% winsorization** to reduce the influence of extreme
observations.

## Reproducibility

To reproduce the analysis:

1.  Obtain a Polygon.io API key.
2.  Download historical stock prices using `DnHistDayPGQuote.java`.
3.  Download earnings data using `BenzingaEarning.java`.
4.  Store the data in MySQL.
5.  Update database settings in the Java source files.
6.  Run `EarningsAnalysis.java`.
7.  Export tables and figures generated by the analysis.
